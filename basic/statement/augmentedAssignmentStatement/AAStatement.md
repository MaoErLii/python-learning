# Augumented Assignment Statement

## combine arithmetic operators in assignments to form an augmented assignment statement

## example

``` 示例

  x += y

  x = x + y

  >>> my_tuple = (10, 20, 30)
  >>> my_tuple += (40, 50, )
  >>> print(my_tuple)

  >>> list_vowels = ['a', 'e', 'i']
  >>>  list_vowels += ['o', 'u']
  >>> print(list_vowels)

```
